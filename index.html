Gentleman

Gentleman is a lightweight projectional editor that allows you to
create and manipulate models on the web, using concepts and projections.

Play with Gentleman Gentleman in action

Features

  • Common features: contextual assistance, copy-paste, undo/redo
  • Support multiple projections for a concept
  • Configurable editor and customizable UI
  • Export format supported: JSON, XML
  • Easy integration with any web application
  • Support multiple notations and multimedia content

Examples

  • Mindmap

    Organize your thoughts as a topic tree: central → main → sub → sub… and assign a marker to any topic for further organization.

  • Traffic Light

    Configure a traffic light system with modes and lights, modeled as a state machine.

  • TodoList

    Create a Todolist to get ready for the day, with many options (tags, recurrence) available to organize your tasks.

  • CMS

    Manage a small data-driven website with a Gentleman powered CMS. It allows you to manage an organization, its members and projects.
